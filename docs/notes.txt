- bipartite graph
  - rule 1: have two disjoint and independent sets (parts) with vertices
  - rule 2: each edge connects two vertices from two sets (parts)
  - eq: does not contain odd length cycles
  - interest only in connected graphs => only one partition
  special labels:
  - balanced: cardinality of those two sets is the same
  - biregular: each vertex have the same degree
  conventions:
  - labels for parts: U, V 
  - label for graph: G = (U, V, E)
